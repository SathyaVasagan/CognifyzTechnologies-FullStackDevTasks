-----------------------------------------------------------------------------------------

**Install the necessary dependencies**

    npm install express ejs body-parser

**Run the Server**

  Start the server:

     node server.js

  or use nodemon:

    nodemon server.js

Open **http://localhost:3000** to see the responsive form.
